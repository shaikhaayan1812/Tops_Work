# SE - Database Management - SQL and PL_SQL

Is folder me Session 1 se Session 9 aur Mini Project 3 ka pura kaam diya hai.

## Kaunsa software use karna hai

- Session 1 se Session 7: MySQL Workbench
- Session 8: Oracle SQL Developer ya Oracle Live SQL
- Session 9: MySQL Workbench
- Mini Project 3: MySQL Workbench

## Kaise use karna hai

1. Har session ka folder kholo.
2. `Questions.txt` me task likhe hue hain.
3. `.sql` file ko software me open karo.
4. Query ek-ek karke run karo.
5. Jaha output aaye, uska screenshot le lena.
6. Screenshot same session folder me rakhna.
7. Pura folder GitHub par upload karna.
8. GitHub folder ka link portal me submit karna.
